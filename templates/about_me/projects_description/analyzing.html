Used:
Language Python3
Libraries mathplotlib, numpy, libtiepie
OS Windows 7
Tool for analyzed USB-oscilloscope "HANDYSCOPE HS3"
Description:

It`s simple script to determine the optimal resonant frequency. This allows to get the efficiency of the voltage source up to 96%.

After running the script, a number of frequencies are fed to the input of the analyzed device, the result is written to a pkl-file for later display.

Example:

After taking waveforms, gain and phase shift are calculated, for further Frequency Response Analysis.
