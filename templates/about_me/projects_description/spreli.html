Description:

Control panel for the voltage source of the welding machine for welding in outer space. The control panel is located inside the space module and it is a backup controller.

The display shows information about the system.

    On the console are control elements:

  • switch for setting the welding current
  • emergency switch off
  • button on/off voltage
  • button on/off filament current
  • button on/off welding current
